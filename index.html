Statiquification du site While42

Le site http://while42.org a été créé :

  • il y a deux ans
  • en 2 heures
  • en html avec du php
  • avec une base mysql

L'idée était :

  • de faire une application ambitieuse
  • pour gérer un annuaire des membres
  • et tout et tout

Mais au final le site a servi à :

  • mettre à jour la liste des chapitres
  • ajouter des liens presse
  • ajouter des photos

L'évolution:

  • pre-générer les pages avec Jekyll
  • héberger le site sur github

Jekyll

  • http://jekyllrb.com
  • ecrit en ruby
  • créé par le fondateur de Github

Github pages

  • http://pages.github.com
  • hébergement gratuit
  • déploiement par git
  • pre-génération avec un hook sur git push

Résultat

  • Mise à jour plus facile
  • Modifs collaboratives par pull-requests
  • Open source et Gratos
